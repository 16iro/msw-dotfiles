# MSW 프로젝트 기본 규칙

- mLua 확장자 작업 시 MSW-MCP 툴을 사용하세요.
- 작업 경로는 `RootDesk/MyDesk` 입니다.
- 생성하는 mLua 파일은 위 경로 하위에 만들어야 합니다.
- MSW 기능 구현 전, `msw-maker-base-skill` 플러그인의 스킬(예: `/msw-general`, `/msw-scripting`)을 먼저 참고하세요.
